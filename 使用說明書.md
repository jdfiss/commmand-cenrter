## 2. 頂部導覽列

常駐在畫面最上方，捲動時不會消失。

### 帳號頭像（左側）
- 點擊頭像 → 開啟使用者切換面板
- 顯示目前登入帳號的名稱與大頭貼

### 壓力指示燈（右側）
- 一個會閃爍的小圓點 + 數字（例如 `2.3h/d`）
- 代表「你每天至少要讀幾小時才能在考試前準備完」
- **顏色代表狀態：**

| 顏色 | 數值 | 意思 |
|------|------|------|
| 🟢 綠色 | < 2.0 h/d | 安全，按部就班即可 |
| 🟡 黃色 | 2.0 – 3.9 h/d | 警告，進度偏緊 |
| 🔴 紅色 | ≥ 4.0 h/d | 高壓，必須清債停止娛樂 |

- 滑鼠**懸停**在燈上 → 顯示詳細狀態訊息

## 3. 隨手記（Inbox）

**用途：** 快速捕捉腦中閃過的任何事，不用當下決定怎麼處理。

### 新增事項
1. 在頂部輸入框輸入任何文字
2. 按 `Enter` 或點「加入」按鈕

### 處理事項（向左滑動）
在手機或支援觸控的裝置上，對事項**向左滑動**，會出現兩個按鈕：

| 按鈕 | 功能 |
|------|------|
| **詳情** | 開啟詳情頁，選擇歸類方向 |
| **刪除** | 直接移除此事項 |

### 詳情頁功能（點詳情後）
跳出 Modal，顯示事項內容，可選擇歸類到：

- **📅 今日任務**：選擇日期 + 時間，自動加入排程，並從 Inbox 移除
- **📖 讀書計畫**：開啟讀書計畫 Modal，手動對應科目
- **⚑ 截止日**：設定截止日期（可加備註），加入截止日管理
- **⏳ 時間債**：估算小時數，手動轉入時間債
- **刪除**：直接移除

> 事項歸類後會自動從 Inbox 移除，不需手動清理。

## 4. 今日心情 & 番茄鐘

### 心情記錄（每日一次）
選擇最符合今天狀態的表情：

| 表情 | 狀態 |
|------|------|
| 😴 | 疲倦 |
| 😟 | 焦慮 |
| 😐 | 普通 |
| 😊 | 良好 |
| 🔥 | 衝動 |

- 點擊表情即儲存
- 右側文字框可輸入一句話記錄，**失焦後自動儲存**

### 番茄鐘計數
- 右側顯示今天已完成幾顆番茄鐘（🍅×n）
- 透過「專注計時器」功能自動累計（詳見第 14 節）

## 5. 今日事項（排程）

顯示**今天**已排定的所有任務，右上角有完成進度條。

### 任務狀態
- **白色圓圈**：未完成
- **橘色圓圈邊框 + 「遲了」標籤**：超過預定時間但未完成
- **綠色打勾**：已完成，文字變灰並加刪除線

### 操作方式

| 操作 | 效果 |
|------|------|
| 點擊任務列 | 切換完成/未完成狀態 |
| 向左滑動 → 編輯 | 修改任務名稱、時間、時長 |
| 向左滑動 → 刪除 | 刪除此任務 |
| 按 ↑↓ 按鈕 | 調整任務順序（hover 才顯示） |
| 按 ⏱ 圖示 | 為此任務啟動專注計時器 |

### 固定課表
- 若今天有固定排課（每週重複），底部會出現「固定課表（n）」
- 點擊展開查看，固定課表**不能在此打勾**，僅供參考

## 6. 長期目標

追蹤備考的核心里程碑，直接影響壓力指示燈。

### 顯示內容
- 目標名稱
- 距截止日天數（近90天變紅，近180天變橘）
- 進度條（完成章數/總章數）
- 已讀/總章數 · 完成百分比 · 截止日期

### 編輯目標
點擊目標右側「**編輯**」按鈕，可修改：
- 目標名稱
- 截止日期
- 已讀章數
- 總章數
- 每章預估小時數（用於壓力計算）

> 修改後壓力指示燈數值會立即更新。

## 7. 截止日管理

追蹤各種非讀書的重要截止日（繳費、報名、作業等）。

### 快速新增
在截止日區塊底部：
1. 輸入名稱
2. 選日期
3. 點「＋」按鈕

### 截止日顏色
| 顏色 | 距離 |
|------|------|
| 🔴 紅色 | ≤ 3 天 |
| 🟡 橘色 | 4–7 天 |
| ⚪ 灰色 | > 7 天 |

### 編輯 / 刪除
向左滑動截止日項目：
- **編輯**：修改名稱、日期、備註
- **刪除**：移除此截止日

### 緊急警告
距今 ≤ 7 天的截止日會在**今日事項區塊下方**額外顯示警告橫幅。

## 8. 時間債務

記錄累積的「未完成工時」。

### 來源
1. **自動產生**：每天午夜 Cron Job 掃描昨天未打勾的任務，依時長轉入
2. **手動加入**：透過隨手記詳情頁 → 時間債

### 顯示
- 右側欄「時間債務」區塊列出所有債務項目與小時數
- 右上角顯示總時數，顏色：
  - > 15h：紅色
  - > 5h：橘色
  - ≤ 5h：灰色

### 清除債務
向左滑動項目 → 點「清除」→ 確認後刪除

### 影響
時間債**直接加入壓力公式**，債越多每天壓力越大。

## 9. 習慣追蹤

每日打卡記錄固定習慣的養成。

### 打卡方式
點擊習慣項目即切換完成/未完成狀態，右側顯示連續天數（🔥n）。

### 本週熱力列
底部小方塊顯示本週每天的習慣完成比例：
- 深灰 = 全部完成
- 中灰 = 完成一半以上
- 淺灰 = 完成一部分
- 白色 = 未完成任何習慣

## 10. 今日記錄（水分 & 睡眠）

### 水分
- 按「＋」「−」記錄今天喝幾杯水（目標 8 杯）
- 底部方格視覺化顯示（藍色 = 已喝）

  - 紅色 < 6h
  - 橘色 6–7h
  - 綠色 ≥ 7h

## 11. 讀書計畫

管理所有備考科目的讀書進度。

### 新增科目
點擊右上角「＋ 新增科目」：
1. 輸入科目名稱
2. 輸入總章數
3. 點「新增」

### 章節操作
每個科目列右側：
- **−** 按鈕：減少已讀章數
- **數字顯示**（已讀/總章）
- **＋** 按鈕：增加已讀章數

### 備註（細項）
每個科目旁有「**備註**」按鈕，點擊展開 inline 文字區：
- 輸入考試範圍、重點提示、讀書節奏等
- **失焦後自動儲存**

### 向左滑動
- **詳情**：開啟科目詳細 Modal
- **刪除**：刪除此科目及所有進度

### 科目詳細 Modal（點科目名稱或詳情）
開啟後包含：

**章節進度格**
- 已讀章節：綠色+刪除線
- 當前章節：藍色+框線
- 待讀章節：灰色
- 點擊章節格 → 自動填入「排入行事曆」的章節欄位

**排入行事曆**

*單日模式：*
1. 選擇日期
2. 選擇時間
3. 輸入章節名稱（選填）
4. 輸入預估分鐘數
5. 點「＋ 加入行事曆」

*跨日模式（點「跨日」切換）：*
1. 選擇開始日
2. 選擇結束日
3. 系統自動在每天排入相同任務
4. 適合連續幾天都要讀同一科目

**備註**：多行文字框，失焦自動儲存

**剩餘章數 & 估算時數**：右下角顯示

### 底部統計
顯示所有科目的整體完成章數和剩餘估算總時數。

### 優先建議
完成率最低的科目會在「今日建議」橫幅中顯示（今日事項下方），並在科目列表中標示「🔥 優先」標籤。

## 12. 本週週報

以 7 欄長條圖顯示本週每天的執行狀況。

| 指標 | 顯示方式 |
|------|------|
| 任務完成率 | 長條高度（藍色=部分，綠色=全完成） |
| 習慣完成數 | 底部小點（黑=完成，白=未完成） |
| 番茄鐘數 | 長條上方 🍅 數字 |

## 13. 行事曆

月視圖行事曆，可新增 / 管理任意日期的任務。

### 月份切換
- **＜ ＞** 按鈕切換月份
- 「今天」按鈕回到今日並開啟今日面板

### 本週快捷列（右側）
顯示本週 7 天縮圖，點擊快速跳到該日面板。

### 格子顏色
- 深藍底 = 今天
- 灰色底 = 選取中的日期
- 灰色文字 = 非本月日期

### 格子內容
每格最多顯示 3 個任務標籤（彩色膠囊），超過顯示 `+n`。
截止日以紅色橫幅顯示（`⚑ 名稱`）。

### 點擊日期 → 開啟日期面板
面板分為三區：

**固定課表**（灰色）：當天有固定課的話顯示在上方

**行程清單**：
- 點擊任務 → 切換完成狀態
  - 今天打勾 → 整頁刷新
  - 其他日期打勾 → 400ms 後自動關閉面板
- Hover 顯示刪除按鈕（X）

**新增任務表單**（面板底部）：

| 欄位 | 說明 |
|------|------|
| 任務名稱 | 必填，Enter 可快速新增 |
| 全天（勾選框） | 勾選後隱藏時間欄，設為全天事件 |
| 開始時間 → 結束時間 | 自動計算 duration（分鐘） |
| 色彩選擇器 | 選任務標籤顏色（10色可選） |
| 加入排程 | 確認新增 |

> 開始時間變更時，結束時間自動 +1 小時。

## 14. 專注計時器（番茄鐘）

### 啟動方式
在今日事項中，hover 任務 → 點右側 **⏱ 圖示**

### 計時器面板
- 預設 25 分鐘倒數（標準番茄鐘）
- **開始** / **暫停** / **繼續** / **↺重設**
- 計時結束：
  - 自動記錄一顆番茄鐘到今日
  - 若瀏覽器允許通知，跳出系統通知
  - 今日心情列右側顯示 🍅×n

### 收起 / 展開
- 計時進行中可點「↙」縮成右下角懸浮按鈕
- 點懸浮按鈕展回完整面板
- 「✕」關閉並停止計時

## 15. 多使用者管理

### 切換使用者
點擊頂部頭像 → 在清單中點選其他使用者 → 自動切換並刷新頁面

### 新增使用者
1. 點頭像 → 點「＋ 新增使用者」
2. 設定頭像（emoji）+ 名稱
3. 按 ✓ 確認
4. 系統自動建立該使用者的獨立資料檔

### 刪除使用者
在使用者清單中，點使用者右側「✕」→ 確認後刪除（資料一併刪除，不可恢復）

> 預設使用者「我」無法刪除。

## 17. 壓力公式說明

```
每日壓力（h/d）= （時間債總時數 + 剩餘讀書時數）÷（距考試天數 × 0.8）
```

- **剩餘讀書時數** = Σ（每科剩餘章數 × 每章小時數）
- **× 0.8** = 假設每天只有 80% 有效學習時間
- 數值越大 = 每天需要讀越多小時

> 壓力計算只取第一個里程碑（`cs_grad_exam`）的截止日。

## 18. 資料自動過夜機制（Cron Job）

`cron_job.py` 需設定為每天午夜（00:00）自動執行。

### 執行內容
1. 掃描昨天排程中所有未完成任務
2. 依時長（分鐘 ÷ 60）轉入時間債
3. 昨天的排程資料保留（不刪除）
4. 確保今天的排程 key 存在

### Windows 設定方式（工作排程器）
1. 搜尋「工作排程器」→ 建立基本工作
2. 觸發程序：每天 00:00
3. 動作：執行 `python D:\temppp\sys\cron_job.py`

### 手動執行
```bash
python D:\temppp\sys\cron_job.py
```
